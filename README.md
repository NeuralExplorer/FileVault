# 📂 FileVault

A modern file management application built using **Python** and **Streamlit**.

## ✨ Features

- 📄 Create files
- 📖 Read file contents
- ✏️ Update existing files
- 🗑️ Delete files
- 🎨 Modern Dark UI
- 📂 Secure workspace management

## 🛠️ Technologies Used

- Python
- Streamlit
- Pathlib
- File Handling

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/NeuralExplorer/FileVault.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run file_manager.py
```
